What is an Array?

An array is a collection of elements stored at contiguous memory locations. Elements can be accessed directly by using an index.

Key Characteristics

  • Indexed: Elements are accessed by their position (0-based indexing)
  • Contiguous: Memory locations are adjacent
  • Fixed Size: Size is determined at creation (in most languages)
  • Random Access: O(1) time complexity for accessing elements

Common Operations

Python Example:
# Create array
arr = [1, 2, 3, 4, 5]

# Access element
print(arr[0]) # Output: 1

# Modify element
arr[2] = 10

# Array length
print(len(arr)) # Output: 5

üìù Strings Basics

What is a String?

A string is a sequence of characters. In many programming languages, strings are immutable, meaning they cannot be changed after creation.

Key Characteristics

  • Character Sequence: Ordered collection of characters
  • Immutable: Cannot be modified after creation (in most languages)
  • Indexed: Characters can be accessed by position
  • Length: Number of characters in the string

Common Operations

JavaScript Example:
// Create string
let str = "Hello World";

// Access character
console.log(str[0]); // Output: H

// String length
console.log(str.length); // Output: 11

// String methods
console.log(str.toUpperCase()); // Output: HELLO WORLD
console.log(str.indexOf("World")); // Output: 6

üìù My Notes & Insights

Add your personal notes, tips, and insights about arrays and strings here:

Key Learning Points

  • Arrays provide O(1) random access but O(n) for insertions/deletions
  • Strings are often treated as character arrays in many algorithms
  • Two-pointer technique is very useful for array problems
  • Sliding window technique works well with arrays and strings

Important Algorithms to Master

  • Two Pointers
  • Sliding Window
  • Binary Search (on sorted arrays)
  • Kadane's Algorithm (for subarray problems)

Common Mistakes to Avoid

  • Forgetting to check array bounds
  • Not handling edge cases (empty arrays, single element)
  • Confusing array indexing (0-based vs 1-based)
  • Not considering space complexity
